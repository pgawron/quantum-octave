MaxMix

Declaration

MaxMix ( [num] )

Description

For a given number 
num
this functions returns maximaly mixed states of 
num
-qubits. If 
num
is ommited then functions returns maximaly mixed state of one qubit.

Example

	octave:1> MaxMix(2)
	ans =

	  0.25000  0.00000  0.00000  0.00000
	  0.00000  0.25000  0.00000  0.00000
	  0.00000  0.00000  0.25000  0.00000
	  0.00000  0.00000  0.00000  0.25000
