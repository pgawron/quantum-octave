Entropy

Declaration

Entropy ( state )

Description

Function 
Entropy
returns von Neuman entropy for density matrix 
state
. For low dimensional systems it can be used as a entanglement measure.

Example

	octave:1> Entropy(MaxMix(2))
	ans = 2
