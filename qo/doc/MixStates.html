MixStates

Declaration

MixStates ( [num,] state [,[num,] state] )

Description

This function allows to mix quantum states. It accepts description of mixture i.e. sequence of optional numbers and matrices and return normalized matrix. It doesn't check if arguments are vailid density matrices.

Example

	octave:1> MixStates(2, State(Ket([0,1])), 3, State(Ket([1,0])))
	ans =
 
	  0.00000  0.00000  0.00000  0.00000
	  0.00000  0.40000  0.00000  0.00000
	  0.00000  0.00000  0.60000  0.00000
	  0.00000  0.00000  0.00000  0.00000
