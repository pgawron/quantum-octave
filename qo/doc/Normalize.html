Normalize

Normalize (ket, num, ket,... )

Description

This function allows to obtain normalized superposition of base vector. It takes sequence of ket vectors and numbers and returns normalized sum of their products num*ket.

Example

	octave:9> Normalize (Ket([0,1,1]),2*Ket([1,0,0]))
	ans =
	
	0.00000
	0.00000
	0.00000
	0.44721
	0.89443
	0.00000
	0.00000
	0.00000
