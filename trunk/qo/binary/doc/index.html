General information

QuantumOctave is a set of octave function for doing quick-and-dirty simulations of quantum computers and -- in general -- quantum systems. You can download the package here. Any sugestions should be directed to authors.

Functions

  • Ket - returns ket vector
  • KetN - returns ket vector
  • State - quantum state represented by density matrix
  • MixStates - mixture of states
  • Normalize - mesurment
  • Id - identity transformation on n qubits
  • H - Hadamard tfansformation on n qubits
  • RotX, RotY, RotZ -
  • Phase -
  • ProductGate - returns gate which is a tensor product of a one-qubit gate and Id gate
  • ControledGate - returns a controled quantum gate
  • CNot - CNot gate
  • Evolve - unitary evolution
  • PTrace - partial tarce
  • Projection - projection
  • PrintAmps - print the amplitudes of state vector
  • PrintProbs - print probabilitis of obtaining any state from the set of base states
  • PlotAmps - plot the amplitudes of state vector
  • PlottAmps - plot probabilitis of obtaining any state from the set of base states
  • BuildBinaryVector - helper function
  • Dec2BinVec - helper function
  • isunitary - helper function
