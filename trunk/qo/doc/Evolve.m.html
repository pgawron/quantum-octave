 2  ## @deftypefn {Function File} {} Evolve (@var{gate},@var{state})
 3  ## This function performs unitary evolution described by gate @var{gate} on the state @var{state}.
 4  ## 
 5  ## For example:
 6  ## @example
 7  ## @group
 8  ## Evolve(ProductGate (2,Not,[1,2]),State (Ket ([0,1])))
 9  ## @result{}
10  ##  0  0  0  0
11  ##  0  0  0  0
12  ##  0  0  1  0
13  ##  0  0  0  0
14  ## @end group
15  ## @end example
16  ## @end deftypefn

21  function ret = Evolve(evolution, state)
22       ret = evolution*state*evolution';
23  endfunction
