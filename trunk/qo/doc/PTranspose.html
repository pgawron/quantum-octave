PTrace

Declaration

PTranspose (state, transpose_qubits)

Description

Function PTranspose allows to perform partial transposuition operation on density matrix state with respect to qubits listed in vector trace_qubits.

Example

	octave:3> s = State ( Normalize (Ket ([0,0,1]) + Ket ([1,0,0])))
	s =
	
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	0.00000  0.50000  0.00000  0.00000  0.50000  0.00000  0.00000  0.00000
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	0.00000  0.50000  0.00000  0.00000  0.50000  0.00000  0.00000  0.00000
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
	
	octave:4> PTranspose (s, [1,2] )
	ans =
	
	0.00000  0.00000  0.00000  0.00000  0.00000  0.50000  0.00000  0.00000
	0.00000  0.50000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000

	0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000  0.00000
