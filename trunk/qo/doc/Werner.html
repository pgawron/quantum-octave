W

Declaration

Werner (param, d)

Description

Function Werner returns density matrix for Werner state in d x d Hilbert space with parameter param. If second argument is ommited it returns Werner state for two qubits.

Example

	octave:10> Werner (.25)
	ans =
	
	0.31250  0.00000  0.00000  0.12500
	0.00000  0.18750  0.00000  0.00000
	0.00000  0.00000  0.18750  0.00000
	0.12500  0.00000  0.00000  0.31250
